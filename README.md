# Garden Shop (Frontend)

Учебный проект интернет-магазина на React.

## Технологии

- React
- Redux Toolkit
- React Router DOM
- React Hook Form
- Vite

## Функционал

- Просмотр категорий
- Просмотр списка товаров
- Страница товара
- Добавление в корзину
- Оформление заказа
- Скидки
- Фильтрация и сортировка

## Запуск проекта

```bash
npm install
npm run dev
Backend
